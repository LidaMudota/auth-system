# Auth System

Простейшее приложение на PHP демонстрирует регистрацию и авторизацию двумя способами: через логин и пароль и через VK OAuth.
В качестве хранилища используется MySQL.

## Структура
- **config/config.php** — настройки БД, VK и путь к логу.
- **db/schema.sql** — схема таблицы `users`.
- **public/** — веб-страницы.
- **src/** — вспомогательные модули.
- **storage/logs/auth.log** — журнал неудачных входов.

## Развёртывание
1. Создайте MySQL‑БД согласно `db/schema.sql`.
2. Укажите параметры в `config/config.php`.
3. Настройте виртуальный хост на каталог `public`.

## Возможности
- регистрация с хешированием пароля;
- вход по логину и паролю с CSRF-защитой;
- вход через VK;
- страница с контентом по ролям.

## Журнал
Все провальные попытки входа фиксируются в `storage/logs/auth.log`.
